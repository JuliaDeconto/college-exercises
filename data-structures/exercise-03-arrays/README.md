# Exercise 03 - Matrix Sum

## Objective
Read a 3x3 matrix of integers from the user, store the values, and calculate the sum of all elements in the matrix.

## Concepts Practiced
- Two-dimensional arrays (matrices)
- Nested `for` loops
- User input with `scanf`
- Accumulating values in a variable
- Output with `printf`
